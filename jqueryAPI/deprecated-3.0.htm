分类: 3.0 版本弃用的 API

所有在此版本的 jQuery 中所弃用的 API。

请参考 http://blog.jquery.com/2016/06/09/jquery-3-0-final-released/ 以获取完整的版本说明和更新日志。

事件 > 事件绑定

.bind()

为一个元素绑定一个事件处理程序。

事件 > 事件绑定

.delegate()

为所有匹配选择器（selector参数）的元素绑定一个或多个事件处理函数，基于一个指定的根元素的子集，匹配的元素包括那些目前已经匹配到的元素，也包括那些今后可能匹配到的元素。

效果 > 自定义 | 特性 > 全局 jQuery 对象的属性

jQuery.fx.interval

该动画的频率（以毫秒为单位）。

事件 > 事件绑定

.unbind()

从元素上删除一个以前附加事件处理程序。

事件 > 事件绑定

.undelegate()

删除当前选择器匹配的所有元素的事件处理程序，根据一组特定根元素的集合。
