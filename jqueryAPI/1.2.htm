分类：版本 1.2

列出了所有在此版本的 jQuery 中新增或修改的 API。
jQuery 1.2 Release Notes

遍历 > 其他遍历

.andSelf()

添加先前的堆栈元素集合到当前组合。

选择器 > 基础筛选 | 选择器 > jQuery 扩展

:animated Selector

选择所有正在执行动画效果的元素.

遍历 > 其他遍历

.contents()

获得每个匹配元素集合元素的子元素，包括文字和注释节点。

效果 > 自定义 | 数据 | 实用工具

.dequeue()

执行匹配元素队列的下一个函数。

属性 | 操作 > Class 属性 | CSS

.hasClass()

确定任何一个匹配元素是否有被分配给定的（样式）类。

选择器 > 基础筛选 | 选择器 > jQuery 扩展

:header Selector

选择所有标题元素，像h1, h2, h3 等.

实用工具

jQuery.inArray()

搜索数组中指定值并返回它的索引（如果没有找到则返回-1）。

实用工具

jQuery.isFunction()

确定的参数是一个Javascript 函数。

实用工具

jQuery.makeArray()

转换一个类似数组的对象成为真正的JavaScript数组。

杂项 > 集合操作 | 表单 | Ajax > 辅助函数

jQuery.param()

创建一个序列化的数组或对象，适用于一个URL 地址查询字符串或Ajax请求。

遍历 > 筛选

.map()

通过一个函数匹配当前集合中的每个元素,产生一个包含的返回值的jQuery新对象。

遍历 > 树遍历

.nextAll()

获得每个匹配元素集合中所有下面的同辈元素，选择性筛选的选择器。

CSS | 位置 | 操作 > Style 属性

.offset()

在匹配的元素集合中，获取的第一个元素的当前坐标， 或设置每一个元素的坐标， 坐标相对于文档。

CSS | 位置 | 操作 > Style 属性

.position()

获取匹配元素中第一个元素的当前坐标，相对于offset parent的坐标。( 译者注：offset parent指离该元素最近的而且被定位过的祖先元素 )

遍历 > 树遍历

.prevAll()

Get all preceding siblings of each element in the set of matched elements, optionally filtered by a selector.

效果 > 自定义 | 数据 | 实用工具

.queue()

显示或操作匹配的元素上已经执行的函数列队。

操作 > DOM 替换

.replaceAll()

用集合的匹配元素替换每个目标元素。

操作 > DOM 替换

.replaceWith()

用提供的内容替换集合中所有匹配的元素并且返回被删除元素的集合。

表单 | Ajax > 辅助函数

.serializeArray()

将用作提交的表单元素的值编译成拥有name和value对象组成的数组。例如[ { name: a value: 1 }, { name: b value: 2 },...]

效果 > 自定义

.stop()

停止匹配元素当前正在运行的动画。

事件 > 事件绑定

.triggerHandler()

为一个事件执行附加到元素的所有处理程序。

操作 > DOM 插入并包裹现有内容

.wrapAll()

在所有匹配元素外面包一层HTML结构。

操作 > DOM 插入并包裹现有内容

.wrapInner()

在匹配元素里的内容外包一层结构。
