分类：操作

此部分中所有的方法是一些操作DOM的方式。 其中几个方法是简单的改变一个元素的一个属性（也列在 属性分类中）， 其他几个设置元素的样式属性的方法 （也列在 CSS 分类中）。 还有一些方法修改整个元素（或一组元素）本身—插入，复制，删除，等等。 所有这些方法被归类为为“setters”（设置器），因为他们改变属性的值。 有些方法 —如.attr(), .html(), and .val() — 还充当“getters”（获取器），从DOM元素检索信息，供以后使用。

属性 | 操作 > Class 属性 | CSS

.addClass()

为每个匹配的元素添加指定的样式类名

操作 > DOM 插入现有元素外

.after()

在元素集合中匹配的每个元素后面插入参数所指定的内容。

操作 > DOM 插入现有元素内

.append()

根据参数设定在每个匹配元素里面的末尾处插入内容。

操作 > DOM 插入现有元素内

.appendTo()

将匹配的元素插入到目标元素的最后面（译者注：内部插入）。

属性 | 操作 > 通用属性

.attr()

获取匹配的元素集合中的第一个元素的属性的值 或 设置每一个匹配元素的一个或多个属性。

操作 > DOM 插入现有元素外

.before()

根据参数设定在匹配元素的前面插入内容（译者注：外部插入）。

操作 > 复制元素

.clone()

创建一个匹配的元素集合的深度拷贝副本。

CSS | 操作 > Style 属性

.css()

获取匹配元素集合中的第一个元素的样式属性的值 或 设置每个匹配元素的一个或多个CSS属性。

操作 > DOM 移除

.detach()

从DOM中去掉所有匹配的元素。

操作 > DOM 移除

.empty()

从DOM中移除集合中匹配元素的所有子节点。

属性 | 操作 > Class 属性 | CSS

.hasClass()

确定任何一个匹配元素是否有被分配给定的（样式）类。

CSS | 尺寸 | 操作 > Style 属性

.height()

获取匹配元素集合中的第一个元素的当前计算高度值 或 设置每一个匹配元素的高度值。

属性 | 操作 > DOM 插入现有元素内

.html()

从集合的第一个匹配元素中获取HTML内容 或 设置每一个匹配元素的html内容。

CSS | 尺寸 | 操作 > Style 属性

.innerHeight()

为匹配的元素集合中获取第一个元素的当前计算高度值,包括padding，但是不包括border。

CSS | 尺寸 | 操作 > Style 属性

.innerWidth()

为匹配的元素集合中获取第一个元素的当前计算宽度值,包括padding，但是不包括border。

操作 > DOM 插入现有元素外

.insertAfter()

在目标元素后面插入集合中每个匹配的元素。

操作 > DOM 插入现有元素外

.insertBefore()

在目标元素前面插入集合中每个匹配的元素。

操作 > Style 属性

jQuery.cssNumber

一个对象，这个包含所有可以不使用单位的CSS属性。.css()方法使用这个对象来确定是否可能附加px到无单位的值。

jQuery.htmlPrefilter()

通过jQuery操作方法修改和过滤HTML字符串。

CSS | 位置 | 操作 > Style 属性

.offset()

在匹配的元素集合中，获取的第一个元素的当前坐标， 或设置每一个元素的坐标， 坐标相对于文档。

CSS | 尺寸 | 操作 > Style 属性

.outerHeight()

为匹配的元素集合中获取第一个元素的当前计算高度值,包括padding，border和选择性的margin。返回一个整数（不包含“px”）表示的值 或 null如果空集合的元素调用。

CSS | 尺寸 | 操作 > Style 属性

.outerWidth()

为匹配的元素集合中获取第一个元素的当前计算宽度值,包括padding，border。

CSS | 位置 | 操作 > Style 属性

.position()

获取匹配元素中第一个元素的当前坐标，相对于offset parent的坐标。( 译者注：offset parent指离该元素最近的而且被定位过的祖先元素 )

操作 > DOM 插入现有元素内

.prepend()

将参数内容插入到每个匹配元素的前面（元素内部）。

操作 > DOM 插入现有元素内

.prependTo()

将所有元素插入到目标前面（元素内）。

属性 | 操作 > 通用属性

.prop()

获取匹配的元素集中的第一个元素的属性（property）值或设置每一个匹配元素的一个或多个属性。

操作 > DOM 移除

.remove()

从DOM中去掉所有匹配的元素。

属性 | 操作 > 通用属性

.removeAttr()

为匹配的元素集合中的每个元素中移除一个属性。

属性 | 操作 > Class 属性 | CSS

.removeClass()

移除集合中每个匹配元素上一个，多个或全部样式。

属性 | 操作 > 通用属性

.removeProp()

为集合中匹配的元素删除一个属性（property）。

操作 > DOM 替换

.replaceAll()

用集合的匹配元素替换每个目标元素。

操作 > DOM 替换

.replaceWith()

用提供的内容替换集合中所有匹配的元素并且返回被删除元素的集合。

CSS | 位置 | 操作 > Style 属性

.scrollLeft()

获取匹配的元素集合中第一个元素的当前水平滚动条的位置或设置每个匹配元素的水平滚动条位置。

CSS | 位置 | 操作 > Style 属性

.scrollTop()

获取匹配的元素集合中第一个元素的当前垂直滚动条的位置或设置每个匹配元素的垂直滚动条位置。

操作 > DOM 插入现有元素内

.text()

得到匹配元素集合中每个元素的文本内容结合，包括他们的后代，或设置匹配元素集合中每个元素的文本内容为指定的文本内容。

属性 | 操作 > Class 属性 | CSS

.toggleClass()

在匹配的元素集合中的每个元素上添加或删除一个或多个样式类,取决于这个样式类是否存在或价值切换属性。即：如果存在（不存在）就删除（添加）一个类。

操作 > DOM 插入并包裹现有内容 | 操作 > DOM 移除

.unwrap()

将匹配元素集合的父级元素删除，保留自身（和兄弟元素，如果存在）在原来的位置。

属性 | 表单 | 操作 > 通用属性

.val()

获取匹配的元素集合中第一个元素的当前值或设置匹配的元素集合中每个元素的值。

CSS | 尺寸 | 操作 > Style 属性

.width()

为匹配的元素集合中获取第一个元素的当前计算宽度值或给每个匹配的元素设置宽度。

操作 > DOM 插入并包裹现有内容

.wrap()

在每个匹配的元素外层包上一个html元素。

操作 > DOM 插入并包裹现有内容

.wrapAll()

在所有匹配元素外面包一层HTML结构。

操作 > DOM 插入并包裹现有内容

.wrapInner()

在匹配元素里的内容外包一层结构。
