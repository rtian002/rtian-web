jQuery.trim()

分类: 实用工具

jQuery.trim( str )返回: String

描述: 去掉字符串起始和结尾的空格。

  • 添加的版本: 1.0jQuery.trim( str )

    • str
      类型: String
      The string to trim.

$.trim()函数会移除字符串开始和结尾处的所有换行符，空格(包括连续的空格)和制表符（tab）。如果这些空白字符在字符串中间时，它们将被保留，不会被移除。
