event.metaKey

分类: 事件 > 事件对象件

event.metaKey返回: Boolean

描述: 表示事件触发时哪个Meta键被按下。

  • 添加的版本: 1.0.4event.metaKey

返回一个布尔值（true 或 false）表示事件触发时那个 META 键是否被按下。在某些平台上此键可以映射到一个替代键名。

在MAC（苹果）系统的键盘上, META 健映射到 Command 健 (⌘)。

在 Windows 系统的键盘上, META 健映射到 Windows 健.
