jQuery.isArray()

分类: 实用工具

jQuery.isArray( obj )返回: boolean

描述: 确定的参数是一个数组。

  • 添加的版本: 1.3jQuery.isArray( obj )

    • obj
      类型: Object
      对象，用来测试是否为一个数组。

$.isArray() 返回一，用来指明对象是否是一个JavaScript数组（而不是类似数组的对象，如一个jQuery对象）。
