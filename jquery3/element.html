返回值:Array<Element(s)>element

概述

根据给定的元素标签名匹配所有元素

参数

elementStringV1.0

一个用于搜索的元素。指向 DOM 节点的标签名。

示例

描述:

查找一个 DIV 元素。

HTML 代码:
<div>DIV1</div>
<div>DIV2</div>
<span>SPAN</span>
jQuery 代码:
$("div");
结果:
[ <div>DIV1</div>, <div>DIV2</div> ]
