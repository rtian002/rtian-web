jQuery.isXMLDoc()

分类: Utilities

jQuery.isXMLDoc( node )返回: Boolean

描述: 检查一个DOM节点是否在XML文档中（或者是一个XML文档）。

  • 添加的版本: 1.1.4jQuery.isXMLDoc( node )

    • node
      类型: Element
      用来检查是否在一个XML文档中的DOM节点。

例子:

检测一个对象是否在一个 XML 文档中。

1
2
 
jQuery.isXMLDoc(document) // false
    jQuery.isXMLDoc(document.body) // false
