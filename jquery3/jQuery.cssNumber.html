jQuery.cssNumber

分类: CSS | Manipulation > Style Properties

jQuery.cssNumber返回: Object

描述: 一个对象，这个包含所有可以不使用单位的CSS属性。.css() 方法使用这个对象来确定是否可能附加px到无单位的值。

  • 添加版本: 1.4.3jQuery.cssNumber

你可以将jQuery.cssNumber想象成一个CSS属性列表，这个列表包含所有可以不使用单位的CSS属性。它用于.css()， 来确定是否需要附加px到无单位的值

jQuery.cssNumber对象的键是驼峰式命名，并且其值都设置成true。如果你想阻止的.css()方法自动添加px单位到特定的CSS属性，你可以添加一个额外的属性到jQuery.cssNumber对象。

1
 
jQuery.cssNumber.someCSSProp = true;

默认情况下，这个对象包含以下属性：

  • zIndex
  • fontWeight
  • opacity
  • zoom
  • lineHeight
  • widows (jQuery 1.6开始添加)
  • orphans (jQuery 1.6开始添加)
  • fillOpacity (jQuery 1.6.2开始添加)
  • columnCount (jQuery 1.9开始添加)
  • order (jQuery 1.10.2开始添加)
  • flexGrow (jQuery 1.11.1开始添加)
  • flexShrink (jQuery 1.11.1开始添加)
